This is the frontend for Friend Knowledgebase (FKB).

## Style Guide

| Element          | Case Format  | Example              |
| ---------------- | ------------ | -------------------- |
| Variables        | `snake_case` | `invoice_amount`     |
| Functions        | `snake_case` | `add_new_task`       |
| File names       | `kebab-case` | `user-profile.ts`    |
| JSON keys        | `kebab-case` | `{ "order-id": "" }` |
| Locale keys      | `kebab-case` | `order-confirmation` |
| React components | `PascalCase` | `UserProfile`        |

> **Note:** `camelCase` is disallowed except for legacy references.
